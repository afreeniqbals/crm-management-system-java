# CRM Management System

A CRM Management System built using Java, JSP, Servlets, JDBC, MySQL, and Bootstrap.

## Features
- User Login Authentication
- Client Management
- Task Management
- Dashboard Analytics
- Search and Filter
- Modern UI

## Technologies Used
- Java
- JSP & Servlets
- JDBC
- MySQL
- Bootstrap
